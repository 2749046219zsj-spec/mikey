┌─────────────────────────────────────────────────┐
│  📦 安装步骤（3步搞定）                         │
└─────────────────────────────────────────────────┘

  1️⃣  打开 chrome://extensions/
  2️⃣  启用"开发者模式"
  3️⃣  加载 chajian 文件夹

┌─────────────────────────────────────────────────┐
│  👤 首次使用                                    │
└─────────────────────────────────────────────────┘

  注册 → 登录 → 开始使用

  测试账号：
  📧 邮箱：test@example.com
  🔑 密码：123456（至少6位）

┌─────────────────────────────────────────────────┐
│  📸 三种上传方式                                │
└─────────────────────────────────────────────────┘

  方式1️⃣  点击插件 → 上传图片按钮

  方式2️⃣  右键网页图片 → 上传到我的图库
         （需登录，数据隔离）

  方式3️⃣  右键网页图片 → 上传到竞品图库
         （无需登录，所有人可见）

┌─────────────────────────────────────────────────┐
│  💾 存储配额                                    │
└─────────────────────────────────────────────────┘

  每用户：100 MB（可在数据库调整）
  单文件：10 MB 以内
  格式：JPG, PNG, GIF, WebP

  💡 删除图片自动释放空间

┌─────────────────────────────────────────────────┐
│  🔒 数据安全                                    │
└─────────────────────────────────────────────────┘

  ✅ JWT Token 认证
  ✅ 数据完全隔离
  ✅ 按用户 ID 分文件夹
  ✅ RLS 权限保护
  ✅ 无法访问他人数据

┌─────────────────────────────────────────────────┐
│  ⚡ 常用操作                                    │
└─────────────────────────────────────────────────┘

  查看图片：点击插件图标
  删除图片：图片下方的删除按钮
  刷新列表：点击刷新按钮
  退出登录：右上角退出按钮

┌─────────────────────────────────────────────────┐
│  🔧 遇到问题？                                  │
└─────────────────────────────────────────────────┘

  问题1：右键菜单没有选项
  解决：chrome://extensions/ → 重新加载插件

  问题2：上传失败
  解决：检查是否登录、文件大小、存储配额

  问题3：看不到图片
  解决：点击刷新按钮

  问题4：注册失败
  解决：密码至少6位，邮箱格式正确
